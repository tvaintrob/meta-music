# meta-music

## Install
```
git clone https://github.com/tvaintrob/meta-music.git
cd meta-music
python setup.py install
```

## Usage
```
usage: meta [-h] [--current-dir] [-o OUT_DIR] [FILE [FILE ...]]

positional arguments:
  FILE           files to query

optional arguments:
  -h, --help     show this help message and exit
  --current-dir  Query all .mp3 files in the current directory
  -o OUT_DIR     output directory
```
